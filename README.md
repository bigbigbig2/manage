## 1.简介

企业管理系统MIS全栈vue3-koa2练习项目，使用通用型架构，包含企业人员管理，系统菜单管理，职位(角色)管理，部门管理，审批流管理，审批流申请，审批流审核等功能。

- `vue3全家桶`
- UI：`elementPlus`
- 服务：`koa2`
- 数据库：`mongodb`
